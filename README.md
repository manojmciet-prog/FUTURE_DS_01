# Business Sales Dashboard – Task 1

# Internship
Future Interns – Data Science & Analytics

# Objective
To analyze e-commerce sales data and create an interactive Power BI dashboard
to identify sales trends, top products, and high-revenue categories.

# Tools Used
- Power BI Desktop
- Excel / CSV
- GitHub

# Dashboard Pages
# Page 1 – Sum Of Total Sales
- Total Sales (Card)
- Sales by Category (Bar Chart)
- Sales by Product (Column Chart)

# Page 2 – Sales Details
- Sales Trend Over Time (Line Chart)
- Detailed Sales Table
- Category Filter (Slicer)

# Key Insights
- Electronics category generated the highest revenue.
- Mobile is the top-selling product.
- Sales show an increasing trend over time.

## Conclusion
This dashboard helps businesses understand sales performance
and make data-driven decisions.
